# UniTools — All-in-One Utility Web App

A unified, privacy-first web platform built with Python Flask.

## Features
- PDF Tools (Convert, Merge, Compress, Protect, Unlock)
- Image Tools (Resize, Compress, Convert, Border)
- Chart Tools (Gantt Chart, Pie Chart)
- AI Resume Matcher (TF-IDF + Cosine Similarity)

## Run Locally
pip install -r requirements.txt
python app.py
